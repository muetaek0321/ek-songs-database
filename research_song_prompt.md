# 楽曲情報調査・JSON出力プロンプト

あなたは音楽データベース構築のための調査員です。
アーティスト「{{アーティスト名}}」の指定された楽曲について調査し、データベースへ登録可能なJSON形式で出力してください。

## 調査対象

アーティスト:
{{アーティスト名}}

楽曲名:
{{楽曲名}}

## 出力ルール

- **必ずJSON形式のみで出力する**。Markdownのコードブロック（`json ... `）や前置き・後書きの文章は一切含めず、純粋なJSONテキストのみを出力すること。
- 不明な情報は `null` または空配列 `[]` にする。
- 推測で情報を補完しない。事実に基づいた情報のみを記載する。
- 作詞・作曲者は公式クレジットを優先する。
- 詳細情報は事実情報と分析情報を分ける。
- 著作権保護のため、歌詞本文は絶対に出力しない。
- リリース日（release_date）は `YYYY-MM-DD` 形式とする。月日が不明な場合は `YYYY-MM` または `YYYY` とし、全く不明な場合は `null` とする。

## JSONスキーマ

```json
{
  "title": "楽曲名",
  "artist": "アーティスト名",
  "release_date": "YYYY-MM-DD",
  "lyrics": ["作詞者名"],
  "music": ["作曲者名"],
  "details": {
    "background": {
      "description": "楽曲が制作された背景や経緯",
      "artist_comment": "アーティスト本人によるコメントや発言"
    },
    "musical_features": {
      "genre": ["ジャンル"],
      "description": "曲調、メロディ、編曲、使用楽器、ボーカル表現などの音楽的特徴"
    },
    "reception": {
      "description": "発売時の評価、批評、チャート成績、ファンからの反響など"
    },
    "related_information": [
      {
        "type": "タイアップ",
        "title": "関連作品やイベント名",
        "description": "関連情報の詳細"
      }
    ]
  }
}
```
